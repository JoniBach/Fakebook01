                                                                                                                                                                                            Dan Jon • 2 years ago

                                                                                                                                                                                            If you've got lines it's because the aspect ratio of your video isn't 16:9. The percentage set for the padding-bottom only works on videos that are 16:9 in aspect ratio. If you had a video that was 4:3 (320 x 240), for example, the percentage would need to be different. To work out the different percentage you would convert 4:3 to a percentage:

                                                                                                                                                                                            (3/4) * 100 = 75%

                                                                                                                                                                                            This would then be the number you'd use as your percentage for padding-bottom.
